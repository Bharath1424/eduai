# **App Name**: EduAI

## Core Features:

- Secure Sign-up and Login: Allows students to create accounts and log in securely to access personalized features.
- AI Chatbot Assistant: An AI-powered chatbot to provide instant study help, answer questions, and offer guidance.
- Reasoning tool: An LLM tool to decide when to inject supporting facts and information into the chatbot responses to ensure learning.
- Customizable Quiz Generator: Enables students to generate quizzes based on specific topics they choose.
- Personal Dashboard: Tracks quiz history and chatbot interactions to help students monitor their progress.

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4) to create a welcoming and trustworthy feel.
- Background color: Clean white (#F9FBFC) for a modern, uncluttered look and enhanced readability.
- Accent color: Sky blue (#74b4ff) to highlight interactive elements and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif suitable for both headlines and body text, ensures excellent readability.
- Use clean, modern icons to represent different study topics and features.
- Maintain a clean and intuitive layout to ensure easy navigation and a user-friendly experience.
- Use subtle animations and transitions to enhance interactivity and provide visual feedback.
